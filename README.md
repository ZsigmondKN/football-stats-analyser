# Football Stats Analyser

![Project Demo](docs/images/Screenshot%202026-06-14%20213314.png)

## Overview

A desktop application for analyzing and visualizing professional football player statistics across multiple European leagues. Features an intuitive graphical interface with advanced filtering, querying, and interactive performance visualization.

## Tech Stack

* Java 11
* Gradle
* JUnit (Testing Framework)
* Swing (GUI Framework)

## Key Features

* Extraction and aggregation of multi-league player datasets
* Advanced query system for filtering by player statistics
* Interactive radar chart visualization for performance analysis
* User-friendly graphical dashboard interface

## How to Run

1. Ensure Java Development Kit (JDK) 11 or higher and Gradle are installed.

2. Run project using: `gradle run`
